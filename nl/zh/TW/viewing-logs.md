# 檢視 Hardware Firewall 的日誌報告
{: #viewing-log-reports-for-hardware-firewall}

日誌可按照每一個 IP 來取得，方法是導覽至受保護的裝置、選取**防火牆**標籤，然後按一下**動作 > 防火牆日誌**。日誌會以 .CSV 格式呈現並包含下列內容：

**事件類型：** 防火牆採取的動作（拒絕）

**通訊協定：** 用於通訊的通訊協定（TCP/PING/UDP/IRD/等等）

**來源 IP 位址：** 封包起源的 IP

**來源埠：** 封包起源的埠

**目的地 IP：** 封包的預期目標

**目的地埠：** 封包的預期埠

**建立日期：** 動作的日期和時間（24 小時制）
